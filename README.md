# react-boilerplate

This project is a boilerplate for ReactJs, including:

- React
- Context
- Hooks (useReducer, useContext)
- State Global using Context + Hooks
- Custom Hooks (Like redux Thunks)
- Axios
- Simple Authentication
- Private-Route
- Styles with SCSS Styling

# Build

Run: ```npm install```

After run: ```npm start```
